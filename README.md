# AlphaAI Robot Control and Programming Experiments

## Abstract

This repository presents a set of programming experiments developed for **AlphaAI educational robots**, focusing on **API-level control**, **remote operation**, and **structured behavior programming**.
The project bridges software engineering and embodied AI by demonstrating how algorithmic logic interacts with physical robotic systems in real-world environments.

---

## 1. Project Motivation

Educational robotics provides an effective platform for understanding how artificial intelligence algorithms are deployed in physical agents.
This project aims to:

* Explore robot control through programmable APIs
* Implement remote interaction mechanisms
* Develop structured robot behaviors via guided tutorials
* Strengthen the connection between AI concepts and real-world robotic execution

---

## 2. Repository Structure

```
api.ipynb                # Low-level robot API interaction and command execution
Remote Control.ipynb     # Remote control and communication-based robot operation
Tutorial2_Code.ipynb     # Tutorial-driven robot behavior programming
README.md                # Project documentation
```

---

## 3. Module Descriptions

### 3.1 API-Based Robot Control (`api.ipynb`)

This module focuses on **direct interaction with the AlphaAI Robot API**, including:

* Establishing robot connections
* Sending motion and action commands
* Executing basic behavior primitives
* Understanding command–response mechanisms

This notebook forms the **foundation layer** for all higher-level robot behaviors.

---

### 3.2 Remote Control System (`Remote Control.ipynb`)

This notebook implements **remote robot operation**, enabling:

* External command input
* Real-time control signal transmission
* Interactive human–robot communication

It highlights key challenges in robotics such as responsiveness, synchronization, and control latency.

---

### 3.3 Tutorial-Based Behavior Programming (`Tutorial2_Code.ipynb`)

This module follows a **guided tutorial structure** designed for learning and experimentation:

* Step-by-step robot behavior construction
* Integration of movement, sensing, and logic
* Reinforcement of robotics programming fundamentals

It emphasizes clarity, reproducibility, and educational value.

---

## 6. How to Run

1. Prepare an **AlphaAI robot** with the corresponding SDK installed
2. Install required Python dependencies
3. Open notebooks using **Jupyter Notebook** or a compatible environment
4. Power on and connect the robot before executing cells sequentially

> ⚠️ Note: Some functions require a physical robot and cannot be fully tested in simulation.
